# bitcoin_price_prediction
# ₿ Bitcoin Price Prediction Using Machine Learning

## Overview

The **Bitcoin Price Prediction Using Machine Learning** project is a web-based application that predicts future Bitcoin prices using historical cryptocurrency market data. The system applies machine learning algorithms to analyze trends, preprocess datasets, train predictive models, and generate price forecasts. It also provides interactive visualizations to help users understand market behavior and make informed decisions.

---

# Features

* User Registration and Login
* Bitcoin Price Prediction
* Historical Data Visualization
* Interactive Price Charts
* Data Preprocessing
* Machine Learning Model Training
* Model Performance Evaluation
* Prediction for Future Dates
* User Dashboard
* Admin Dashboard
* CSV Dataset Upload
* Responsive Web Interface

---

# Objectives

* Predict future Bitcoin prices using machine learning.
* Analyze historical cryptocurrency market trends.
* Improve prediction accuracy through data preprocessing.
* Visualize Bitcoin price movements.
* Provide users with data-driven insights for market analysis.
